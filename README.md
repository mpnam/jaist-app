# jaist-app
Jaist App is built to improve the mobile experience of students who study in Jaist.
### Features ###
1. Hokutetsu Railroad Ishikawa line's timetable (Tsurugi - Nomachi)
2. Jaist Shuttle Bus (Tsurugi Line)
